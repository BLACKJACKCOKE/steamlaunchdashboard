Section 02
WL 목표 KPI & 전환 시뮬레이터
3단계 Flywheel 작동 구조 · P Premium vs F2P WL 비교 차트 · Launch PCCU Top 5 진입 전환 시뮬레이션
Stage 1 — Pre-launch
Wishlist 최대화 = 론칭 당일 이메일 발송 규모 결정.
WL N명 → 론칭 이메일 N건 발송 → D+1 CCU 결정 → Steam 알고리즘 초기 점수. WL이 없으면 론칭 이메일도, 알고리즘 점수도 없음.
Stage 2 — 공개 단계
긍정 Review 최대화 = 구매 전환 가속기.
리뷰 40% 미만 시 알고리즘 피처링 배제. 론칭 초기 72시간이 장기 포지션 결정. SNF Demo 긍정 반응 70%+ 목표.
Stage 3 — Post-launch
플레이·구매 최대화 = 노출 가속기.
높은 CCU → Top 순위 유지 → 오가닉 유입 → Flywheel 가속. PCCU 55K+(Top 5 최소) / 67K+ 목표. The Finals 268K→11K 반면교사.
위시리스트 누적 목표 — ■ Premium vs ■ F2P
CBT 단계(2027.04)부터 분기 · 누운 막대 그래프 · 엑셀 원본 데이터
WL → Launch PCCU 경로 — P · F2P 트랙별 트래킹
⚠️ 두 BM의 핵심 전환 지표는 분모/분자가 다름 — Premium "WL→구매(15%)" ↔ F2P "WL→설치(~42%)"는 직접 % 비교 불가. 공통 비교 지표는 최하단 「WL→PCCU Peak 전환율」.

F2P 전환 특성: 결제 장벽 없어 설치 전환율 높음(~42%). 단, 무료 유저는 이탈이 빠름 — D+7 리텐션 25%로 낮음. 수익은 과금 유저(~10%) IAP에서 발생.
✅ 직접 비교 가능한 단일 지표 — WL → PCCU Peak 전환율

F2P 2,400,000 WL → 58,000 PCCU = 2.42%
→ Premium이 단위 WL당 PCCU 약 1.7× 효율 (D+7 리텐션 35% vs 25% 격차에서 발생). 단, 절대 PCCU는 F2P 트래픽 규모로 보완 가능.
